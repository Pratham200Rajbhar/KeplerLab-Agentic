The Random Forest classifier was trained on the Titanic survival dataset with the following preprocessing and evaluation:

- **Feature engineering**: Sex and Embarked columns were encoded (likely one-hot or label encoding), missing values were handled (imputation or removal).
- **Train/test split**: 80/20 ratio.
- **Performance metrics**:
  - Accuracy: **0.8000**
  - Precision: **0.8333**
  - Recall: **0.6250**
  - F1-score: **0.7143**

The following artifacts were generated and are available for download:

- **confusion_matrix.png**: Heatmap visualization of the confusion matrix, showing true positives, false positives, true negatives, and false negatives.
- **feature_importance.png**: Bar chart ranking features by their importance in the Random Forest model (e.g., Sex, Pclass, Age, Fare).
- **model.pkl**: Serialized Random Forest model for reuse or further analysis.
